# Password-Generator

A modern and responsive Password Generator built using **HTML, CSS, and JavaScript**.  
This application allows users to generate secure passwords with customizable options and real-time strength analysis.

---

##  Features

- Generate secure random passwords
- Customize password length (6–24 characters)
- Include/exclude:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Real-time password strength indicator (Weak / Medium / Strong)
- Copy to clipboard functionality
- Clean and responsive UI

---

##  Tech Stack

- HTML5
- CSS3
- JavaScript (DOM Manipulation, Events, Regex)
